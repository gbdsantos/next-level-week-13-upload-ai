## 💡 About Project

Projeto realizado durante o evento Next Level Week da Rocketseat nos dias 11, 12 e 13 de setembro de 2023.

<br>

## ⚙️ Executar Projeto

```Bash
# Instalar dependências
npm install

# Executar back-end
cd server
npm run dev

# Executar front-end
cd web
npm run dev
```
